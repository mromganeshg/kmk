# களஞ்சிய மேலாண்மை கருவி 
  இந்த GitHub பதிவு, Spark architecture ஐ பயன்படுத்தி எவ்வாறு ஒரு பொருட்களஞ்சியத்தை நிர்வகிக்கப்பது என்பது பற்றியதாகும். 

# களஞ்சியம் (ஆங்கிலம்-Warehouse)
களஞ்சியம் என்பது பலவித பொருட்களை உட்கொண்ட கூடாரம் ஆகும். 
உதாரணமாக
* மருத்துவ பொருட்கள்
* மகிழுந்து உதிரி பாகங்கள்
* கைபேசி பாகங்கள்

மற்றும் பல.

ஒரு களஞ்சியத்தில் ஒரு ஒரு நிமிடமும் பல பொருட்கள் வருவதும் போவதுமாய் இருக்கும்.
ஒரு நாளில் பல நூறு பரிவர்த்தனைகள் நடைபெறும் இடம் அது.
இதனை ஒரு மாதம் ஒரு வருடம் என நீண்டால் அது மிக பெரிய தகவல்களை கொண்ட ஒரு தகவல் களஞ்சியம் ஆகி இருக்கும்.
இவ்வாறான ஒரு பொருள்-தகவல் களஞ்சியத்தை மேலாண்மை செய்வது பாரம்பரிய முறியள் கடினம்.
எனவே இதனை கையாளவே Spark பயன்படுகிறது.
இப்பொழுது என்ற என்ன என்பதை காண்போம்.
