# களஞ்சியம் (ஆங்கிலம்-Warehouse)
களஞ்சியம் என்பது பலவித பொருட்களை உட்கொண்ட கூடாரம் ஆகும். 
உதாரணமாக
* மருத்துவ பொருட்கள்
* மகிழுந்து உதிரி பாகங்கள்
* கைபேசி பாகங்கள்

மற்றும் பல.

ஒரு களஞ்சியத்தில் ஒரு ஒரு நிமிடமும் பல பொருட்கள் வருவதும் போவதுமாய் இருக்கும்.
ஒரு நாளில் பல நூறு பரிவர்த்தனைகள் நடைபெறும் இடம் அது.
